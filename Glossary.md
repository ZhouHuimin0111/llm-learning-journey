# 核心名词解释库
本文档维护学习过程中遇到的专有名词及其解释，保持持续更新。

* **LLM (Large Language Model，大语言模型)**：
* 拥有庞大参数量（通常在数十亿以上）的深度学习模型，通过海量文本训练，能够理解和生成人类语言。
* **RAG (Retrieval-Augmented Generation，检索增强生成)**：
* 一种结合信息检索和文本生成的技术方案。在生成回答前，系统先从外部知识库检索相关信息，作为背景资料输入给模型，旨在减少幻觉 (Hallucination) 并提供实时、准确的信息。
* **Token (词元)**：
* 模型处理和生成文本的最小基本单位。它可以是一个完整的词、词的一部分，或者在某些语言中是一个字符。
* **Embedding (向量化/嵌入)**：
* 将非结构化文本转化为包含语义信息的高维数值向量的过程。这使得计算机能够通过数学计算（如余弦相似度）来判断文本之间的语义相关性。
